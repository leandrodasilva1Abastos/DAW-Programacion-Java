# DAW - Programación Java

Prácticas y proyectos de Java realizados durante 1º de DAW en IES Abastos.

## Proyectos

### TiendaGUI-Propia
Aplicación de tienda con interfaz gráfica en Java Swing.
Desarrollada de forma independiente.

### TiendaGUI-IA
Misma aplicación desarrollada con ayuda de IA.
Usa el patrón MVC (Modelo-Vista-Controlador).

## Tecnologías
- Java
- Swing
- NetBeans
